# Corso Python per Unicredit - ediz. 1 (giugno 2026)

Questo è il repository ufficiale del corso Python per Unicredit - 1a edizioni.

Per scaricare i file premere il bottone verde "Code" in alto a destra e poi selzionare l'ultima voce del menù "Download ZIP".

E' anche possibile scaricare singolarmente i vari file, in  questo modo: 1. click sul file specifico --> il file viene visualizzato
a schermo pienio; 2. premere l'icona "freccia in giù" in alto a destra.

E' importante che, una volta che avete scaricato i file sulla vostra cartella "Downloads" del vostro PC e li abbiate unzippati, spostiate TUTTI i 
file scaricati nella STESSA cartella del vostro PC (quella nella quale lavorerete).

Come individuare i file che sono stati modificati rispetto alla precedente giornata del corso? Tramite la seconda colonna del repo.
Ad esempio, se il docente ha modificato uno o più file del repository al termine della prima giornata, l'etichetta (seconda colonna) del/dei file riporterà:
"Modifica DOPO la PRIMA giornata".

Il repository github è solo un ambiente di transito e non di edit dei file. E' possibile editare (cioè aprire tramite click) il singolo file pere esaminarne i contenuti
ma NON è il modo corretto. Cioè funziona ma con molti limiti (ad esempio i file software NON sono eseguibili).

Questo file README.md è il canale di comunicazione il docente e i partecipanti al corso tra una giornata e l'altra. Proprio perquesto è l'UNICO file che, all'apertura 
del repository github è aperto e visualizzato a schermo pieno.
